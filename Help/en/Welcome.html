Welcome to AutoHotFlow

Why AutoHotFlow?

AutoHotFlow allows you to ease up your work flow, you can create little applications that do whatever you want.

What is possible with AutoHotFlow?

AutoHotFlow can perform simple actions, like doing some calculations, moving windows, showing notifications and moving files, and there are also some powerful actions, like simulating user input, speech output, download file. More powerful actions are coming…
AutoHotFlow has triggers that allows to create flows that do their work automatically.

Do I need foreknowledge?

You don't need any programming skills. You can start right now and succeed! If you want to learn programming, this tool is the right one to get in touch with computational logic.

How do I start?

Open the manager and look inside some demonstration flows.
Create new flow and name it. Open it and start adding new elements and connecting it. It's intuitive!

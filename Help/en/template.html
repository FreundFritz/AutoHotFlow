Input string

String that will be converted to lowercase or uppercase.
A variable's content can be inserted by enclosing them into percent signs.

Source file

The name of a single file or folder, or a wildcard pattern.

Both asterisks and question marks are supported as wildcards. A match occurs when the pattern appears in either the file's long/normal name or its 8.3 short name.
The file pattern is assumed to be in the working directory if an absolute path isn't specified.

File path

Name and path of the file. The file path is assumed to be in the working directory if an absolute path isn't specified.



Supplied variables

A_Index

contains the number of the current iteration. It starts with the value 1 and increases on every loop.


Examples:

not 5 < 3 (results to 1)

a_index <= 5

Note!

Note!

This might cause trouble if you delete or add an element inside the loop. For example:

Example:

A list contains following elements: ONE, TWO, THREE

On first iteration a_LoopValue contains ONE. Inside the loop, an action is used, that deletes the first element. On second iteration a_LoopValue will contain THREE. After that iteration the loop will end.

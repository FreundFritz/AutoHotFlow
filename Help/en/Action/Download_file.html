Dowload file

The action Download file downloads a file from the internet.

Settings

URL

URL of the file to download. For example, http://someorg.org might retrieve the welcome page for that organization.

File path

Name and path of the file where the download will be written.
The file path is assumed to be in the working directory if an absolute path isn't specified.

Examples:

Delete one file from MyDocuments folder.

File pattern: %A_MyDocuments%\Example.txt

Delete all files from a drive.

File pattern: G:\*

Delete all text files from the working directory.

File pattern: *.txt

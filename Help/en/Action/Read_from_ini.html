Read from ini

The action Read from ini reads a value from a standard format .ini file.

Settings

Output variable name

Name of the variable which will contain the result.

Path of .ini file

Name and path of the file.
The file path is assumed to be in the working directory if an absolute path isn't specified.

Action

If Read a key is checked, only one key will be read and its value will be stored to the output variable.

If Read a section is checked, a list of all section names will be stored to the output variable..

Section and Key

The section and the key that contain the value.

Behavior on error

If the file, section or key won't be found, the action could throw an exception or write a default value in the variable.

Default value on failure

If Insert default value in the variable is selected, in case of an error, this value will be stored to the variable.

Remarks

A standard ini file looks like this:

[SectionName]
Key=Value
